# Changelog

All notable changes to **MAGENAIS** are documented in this file.

This project follows the principles of **Keep a Changelog** and **Semantic Versioning (SemVer)**.

---

## Versioning

MAGENAIS uses the following version format:

```
MAJOR.MINOR.PATCH
```

Example:

```
2.3.1
```

Where:

- **MAJOR** — Breaking architectural changes
- **MINOR** — New features and improvements
- **PATCH** — Bug fixes and small enhancements

# [1.0.0] — Initial Release

## Initial Prototype

The first public prototype of MAGENAIS.

### Features

- Browser-based interface
- Chat interface
- Initial provider integrations
- Prompt management
- Basic AI generation
- HTML-based implementation

### Notes

This version served as the proof of concept that demonstrated the feasibility of a browser-first AI application.

Future releases migrate this architecture into the new modular platform.
