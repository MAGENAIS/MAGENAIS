# Examples

Example workflows and small end-to-end projects demonstrating how to compose
MAGENAIS nodes. Load a workflow JSON file from
[`workflows/`](workflows/) through the workflow editor to try it.

| Example | Description |
|---|---|
| [`workflows/summarize-and-narrate.json`](workflows/summarize-and-narrate.json) | Research a topic across academic sources, then narrate the synthesized summary as speech. |

The in-app **Workflow** button (top right) provides a UI for building chains like this yourself — pick step types, chain their outputs, and run them — without hand-editing JSON.

See [`docs/workflow/WORKFLOW_ENGINE.md`](../docs/workflow/WORKFLOW_ENGINE.md)
and [`docs/workflow/NODE_TYPES.md`](../docs/workflow/NODE_TYPES.md) for the
full node type reference.
