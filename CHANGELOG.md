# Changelog

All notable changes to MAGENAIS are documented here in summary form.
For the complete version history and detailed release notes, see
[`docs/CHANGELOG.md`](docs/CHANGELOG.md).

## [2.1.0] — Current

- Migrated from a single-file monolithic build to a modular TypeScript/Vite
  architecture (kernel, provider registry + router, workflow engine, plugin
  system).
- Full provider fallback chain across 58 providers spanning text, image,
  video, audio, speech, music, and transcription.
- Ten generation modes: Text, Image, Video, Audio, Data, Documents, Research,
  Game Generator, Agents, and Help.
