# Roadmap

This is a short summary. The full phased roadmap lives in
[`docs/ROADMAP.md`](docs/ROADMAP.md).

- **Now** — Stabilize the modular architecture, expand provider and plugin
  coverage, and improve the workflow/graph editor.
- **Next** — Marketplace for community plugins and providers, richer
  multi-node workflow authoring, deeper project/asset management.
- **Later** — MAGENAIS as a full AI Operating System: distributed compute,
  team collaboration, and a mature extension ecosystem.

Have a feature request? Open an issue using the feature request template
under [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE/).
