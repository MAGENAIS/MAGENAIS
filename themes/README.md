# Themes

A theme is a CSS file that redefines the design tokens declared in
[`src/css/variables.css`](../src/css/variables.css). See
[`docs/ui/THEMES.md`](../docs/ui/THEMES.md) for how themes are registered and
switched at runtime through the Theme system.
